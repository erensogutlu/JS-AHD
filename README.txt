/* JS-AHD | Anlık Hava Durumu */

Bu proje, JavaScript kullanılarak geliştirilmiş modern ve interaktif bir hava durumu uygulamasıdır. Şehir araması yapabilir, favori şehirlerinizi kaydedebilir ve hava durumlarını şık kartlarda görebilirsiniz.

* Özellikler :
+ Şehir ismine göre hava durumu sorgulama
+ Basit ve şık kullanıcı arayüzü
+ Gerçek zamanlı veri çekme (API kullanımı)
+ Favori şehirleri localStorage ile saklama
+ Kart silme özelliği
+ Mobil uyumlu tasarım

* Kullanılan Teknolojiler :
+ HTML / CSS
+ JavaScript

* Geliştirici : Eren Söğütlü

LİNK : https://erensogutlu-hava-durumu-web.netlify.app/

/* JS-AHD | Current Weather */

This project is a modern and interactive weather application developed using JavaScript. You can search for cities, save your favorite cities and see the weather conditions on stylish cards.

* Features :
+ Weather query by city name
+ Simple and stylish user interface
+ Real-time data retrieval (API usage)
+ Storing favorite cities with localStorage
+ Card deletion feature
+ Mobile compatible design

* Technologies Used :
+ HTML / CSS
+ JavaScript

* Developer : Eren Söğütlü

LINK : https://erensogutlu-hava-durumu-web.netlify.app/
